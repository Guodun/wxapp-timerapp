# 微信小程序－辩论赛计时器

## 说明
- 实现辩论赛计时器功能;
- 可以设置不同阶段是否启用、时间限制及提前多少秒语音提示;
- 涉及动画实现、JS定时器、数据缓存、面向对象编程以及表单相关组件。
